Breakthrough Performance. It'll blow your mind.

Samsara offers break-through real-time performance with the ability to process and index terabytes of data. Samsara API handles around 1500 HTTP requests per second, ingesting over 175GB of data into Kafka per day. Samsara core can process 550 million events a day.

Clients for many languages.Start publishing events right away.

Samsara provides clients for various languages and platforms allowing you to focus on your functionality. Wer are always adding new clients, so if you would like a client for the language you use, just drop us a note.

Out of the box deployment templates for the cloud. You can have your cake and eat it too !

Samsara is fully cloud native. Samsara provides out-of-the-box deployment templates for Amazon EC2, Amazon ECS and Kubernetes platforms.
